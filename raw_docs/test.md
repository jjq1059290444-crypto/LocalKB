# 开关电源基础

## Buck 电路

Buck 电路是一种降压型 DC-DC 转换器。其基本工作原理是通过开关管（通常是 MOSFET）的周期性导通和关断，将输入电压降低到所需的输出电压。

输出电压公式：Vout = D × Vin，其中 D 是占空比。

## Boost 电路

Boost 电路是一种升压型 DC-DC 转换器。当开关管导通时，电感储存能量；当开关管关断时，电感释放能量，与输入电压叠加后输出。

输出电压公式：Vout = Vin / (1 - D)

## 关键参数

- 开关频率通常在 100kHz 到 2MHz 之间
- 效率一般可达 85%-95%
- 纹波电压应小于输出电压的 1%
