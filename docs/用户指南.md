# LocalKB 用户指南

## 安装与启动

### 系统要求

- Windows 10 / 11（推荐），Linux 和 macOS 也可运行
- Python 3.10 或更高版本
- 至少 4GB 可用内存
- 建议使用 NVIDIA GPU（非必需，CPU 亦可运行嵌入模型）

### 安装步骤

1. 下载或克隆项目代码
2. 双击运行 `setup.bat`，自动创建虚拟环境并安装依赖
3. 双击运行 `启动.bat` 启动应用

### 手动安装（如 setup.bat 失败）

```bash
cd LocalKB
python -m venv venv
venv\Scripts\pip install -r requirements.txt
venv\Scripts\python desktop_app\main.py
```

## API 配置

首次使用需要配置大模型 API 连接。

### DeepSeek（推荐，国内用户）

1. 访问 [platform.deepseek.com](https://platform.deepseek.com/) 注册
2. 在「API Keys」页面创建 Key
3. 在 LocalKB 的「API配置」页面：
   - 供应商选择 **DeepSeek**
   - 粘贴 API Key
   - 点击「测试连接」确认
   - 点击「保存配置」
4. 重启应用

### OpenAI

1. 访问 [platform.openai.com](https://platform.openai.com/) 获取 API Key
2. 供应商选择 **OpenAI**，输入 Key 即可

### 自定义 API（如 Ollama 本地模型）

1. 供应商选择 **Custom**
2. 填入你的 API 地址（如 `http://localhost:11434/v1`）
3. 填入模型名（如 `llama3`）

## 知识库管理

### 支持的格式

- `.md` — Markdown 文档
- `.txt` — 纯文本文档

### 上传文档

**方式一**：点击上传区域，在文件选择器中选择文件

**方式二**：从文件管理器直接拖拽文件到上传区域

### 处理流程

文件上传后，系统会自动执行四步处理：

1. **解析** — 读取文件内容，去除 YAML 前言
2. **切片** — 按标题、段落将文档切分为重叠的片段（每段 ~350-700 字符）
3. **嵌入** — 使用嵌入模型将每个片段转为向量
4. **存储** — 将向量和元数据存入 ChromaDB

### 查看与管理

- 页面下方显示已索引的文档和切片总数
- 点击「刷新列表」更新统计
- 点击「清空索引」可重置整个知识库

## 对话问答

### 提问

在底部输入框中输入问题，按 **Ctrl+Enter** 或点击「发送」按钮提交。

### 回答流程

1. 系统检索与问题最相关的文档片段（混合 ChromaDB + BM25）
2. 将检索到的片段和问题组装成 Prompt
3. 调用 LLM 生成回答（流式显示）
4. 右侧面板显示参考来源

### 回答质量建议

- 问题尽量具体，避免过于宽泛
- 上传的文档内容越丰富，回答质量越高
- 可在参数设置中调整 System Prompt 来改变回答风格

---

## 参数设置

| 参数 | 说明 | 默认值 |
|------|------|--------|
| Top-K | 检索返回的文档片段数量 | 10 |
| Temperature | LLM 生成温度（0=确定，1=随机） | 0.3 |
| System Prompt | 系统提示词，定义 AI 的行为和回答风格 | 见默认值 |
| Embedding Model | 嵌入模型选择 | bge-small-zh-v1.5 |

### 嵌入模型说明

- **bge-small-zh-v1.5**：中文优化，512维向量，适合中文知识库
- **all-MiniLM-L6-v2**：英文优化，384维向量，适合英文知识库

## 常见问题

### Q: 启动报错 "KMP_DUPLICATE_LIB_OK"

系统已自动设置此环境变量。如仍有问题，请确认没有其他程序占用 PyTorch 库。

### Q: 上传文档后没有切片

检查文档是否为 `.md` 或 `.txt` 格式，且文件编码为 UTF-8。

### Q: 提问后回答很慢

- 首次提问需要加载嵌入模型（可能需要几十秒）
- 后续提问会快很多
- 检查网络连接是否正常

### Q: 如何备份数据

复制 `data/` 目录即可备份所有知识库数据和历史记录。

### Q: 如何更换 API 供应商

在「API配置」页面修改供应商和 Key，保存后重启应用即可。
