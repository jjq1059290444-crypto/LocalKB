RAG 检索增强生成流程

从提问到回答，四步走通
01

📥 文档导入 & 向量化

上传 Markdown / PDF / DOCX / PPT 等文档，自动解析、结构切片，BGE-M3 生成稠密 + 稀疏双向量，存入 Qdrant。

▼
02

🔍 混合检索

同时进行稠密语义搜索 + 稀疏关键词搜索（BM25 词法权重），Qdrant 内置 RRF 算法融合排序，可选 Cross-encoder 二次重排。

▼
03

📋 上下文构建

检索到的 Top-K 片段 + System Prompt + 多轮对话历史 → 拼接为完整上下文，注入 LLM 请求。

▼
04

🤖 LLM 生成答案

通过 OpenAI 兼容 API 调用大模型（DeepSeek / GPT-4o / 自定义），基于参考材料流式生成回答，附带引用来源。
