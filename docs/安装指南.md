# LocalKB 安装指南

## 第一步：安装 Python 3.12

下载 Python 3.12.10（64-bit）：

https://www.python.org/ftp/python/3.12.10/python-3.12.10-amd64.exe

双击运行安装程序，**务必勾选**：

- ✅ **Add Python to PATH**（一定要勾！）

然后点击 Install Now，等待安装完成。

验证是否安装成功：按 `Win + R`，输入 `cmd` 回车，输入：

```
python --version
```

如果显示 `Python 3.12.10` 则安装成功。

---

## 第二步：解压 LocalKB

将 `LocalKB.zip` 解压到任意目录（建议路径不含中文和空格）。

---

## 第三步：启动

进入解压后的文件夹，双击 **启动.bat**。

首次启动会自动：
1. 创建虚拟环境
2. 安装依赖（约 2-3 分钟，需要网络）
3. 进入引导设置页面

后续启动直接进入主界面。

---

## 第四步：引导设置

1. 选择语言
2. 配置 LLM（API 地址 + Key + 模型）
3. 选择嵌入模型（默认 bge-small-zh-v1.5 已内置，无需下载）
4. 完成 → 进入主界面

---

## 常见问题

### Q: 启动闪退？
确保已安装 Python 3.12 并勾选了 "Add Python to PATH"。

### Q: 下载嵌入模型失败？
默认内置了 bge-small-zh-v1.5，无需额外下载。如需 BGE-M3 等大型模型，建议网络畅通时在设置页面下载。

### Q: 没有 API Key？
推荐使用 [DeepSeek](https://platform.deepseek.com/) 注册获取（便宜好用）。也可以用 OpenAI 或其他兼容接口。
